Features of this software

Functions of the software :
Text to postscript and PDF,image conversion,custom scripting,batch 
processing,support custom component plugins.A front-end for page
formating with enscript with page preview. Batch processing ability. 
  
Features through a configuration file :
a)Create your own menu,menu sub-items and speed-buttons
b)Run your own program or scripts from the created menu and speed-buttons
c)Build your own VCL component and plug it into the programs
d)Suitable for files batch conversion,batch processing,batch printing
  and networking
e)Configure the user interface easily  
f)Can be use as a text editor
